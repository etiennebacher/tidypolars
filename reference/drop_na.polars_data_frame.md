# Drop missing values

By default, this will drop rows that contain any missing values. It is
possible to specify a subset of variables so that only missing values in
these variables will be considered.

## Usage

``` r
# S3 method for class 'polars_data_frame'
drop_na(data, ...)

# S3 method for class 'polars_lazy_frame'
drop_na(data, ...)
```

## Arguments

- data:

  A Polars Data/LazyFrame

- ...:

  Any expression accepted by
  [`dplyr::select()`](https://dplyr.tidyverse.org/reference/select.html):
  variable names, column numbers, select helpers, etc.
