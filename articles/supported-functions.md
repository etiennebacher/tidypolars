# List of supported functions

This document lists base R and `tidyverse` functions fully or partially
supported by `tidypolars`. It complements the
[Reference](https://www.tidypolars.etiennebacher.com/reference/) page:
the reference page shows `data.frame`-level functions while this page
shows functions that can be used in
[`filter()`](https://dplyr.tidyverse.org/reference/filter.html),
[`mutate()`](https://dplyr.tidyverse.org/reference/mutate.html), etc.

The “Notes” column highlights possible differences in the output of the
original functions and of their translation by `tidypolars`.
